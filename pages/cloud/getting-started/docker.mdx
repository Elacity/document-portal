# Docker Deployment

<Callout type="warning">
**Coming Soon** — Docker support is not yet available. We're working on an official Docker image and will update this page when it's ready.
</Callout>

Docker deployment is on the roadmap and will provide containerized deployment for easy setup and isolation. In the meantime, use one of the available installation methods:

- [Local Install](/cloud/getting-started/local) — Easiest way to try PC2
- [VPS Server](/cloud/getting-started/vps) — Always-on, recommended for 24/7
- [NVIDIA Jetson](/cloud/getting-started/jetson) — GPU-accelerated private AI
- [From Source](/cloud/getting-started/source) — For developers

---

Want to be notified when Docker support launches? Follow the [GitHub repository](https://github.com/Elacity/pc2.net) for updates.
